Send to focused window
Sent when activating (turning ON)
Sent when deactivating (turning OFF)
Toggle Examples
  • Sneak Attack Toggle:
    On: input /ja "Sneak Attack" <me>
    Off: input /echo SA Ready
  • GearSwap Mode:
    On: gs c set MeleeMode Acc
    Off: gs c set MeleeMode Normal
  • Follow Toggle:
    On: input /follow <t>
    Off: input /heal
Targeting
  • Focus Mode ON (default): Command is sent to whichever FFXI window currently has focus.
  • Focus Mode OFF: Command executes directly on the server character, even without focus.
  • Character name: Overrides Focus Mode. The command is sent to the named character via IPC. Use @all to send to every character.
