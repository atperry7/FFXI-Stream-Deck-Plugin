Command Examples
  • input /ma "Cure" <t> - Cast spell
  • input /ja "Provoke" <t> - Use ability
  • input /ws "Savage Blade" <t> - Weapon skill
  • send Whitey input /ma "Cure IV" <me> - Send to alt
  • lua r gearswap - Reload addon
  • gs c toggle - GearSwap command
Targeting
  • Focus Mode ON (default): Command is sent to whichever FFXI window currently has focus.
  • Focus Mode OFF: Command executes directly on the server character, even without focus. Useful for commands like switch_focus.
  • Character name: Overrides Focus Mode. The command is sent to the named character via IPC. Use @all to send to every character.
